주요 기능

📂

워크스페이스 관리

세로 탭 사이드바에서 여러 프로젝트를 한눈에 관리. Git 브랜치, CWD, PR 번호, 리스닝 포트 자동 표시.

🔔

에이전트 알림

OSC 9/99/777 시퀀스 감지. 에이전트가 입력을 기다리면 알림 링으로 즉시 표시.

🌐

내장 브라우저

터미널 옆에 WebView 브라우저 패널. JavaScript 실행 콘솔 내장.

▧

패널 분할

가로/세로 분할, 드래그 리사이즈. 각 패널에 독립적인 이름 지정.

🔄

세션 복원

껐다 켜도 워크스페이스, 패널 레이아웃, 터미널 화면 내용 그대로 복원.

📡

CLI 제어

Named Pipe API로 외부에서 실시간 제어. wumx notify, wumx list-workspaces 등.

🔗

컨텍스트 공유

워크스페이스/패널 간 키-값 데이터 공유. TTL 지원.

⚡

GPU 가속

xterm.js WebGL 렌더러로 62fps 렌더링. 대량 출력에도 끊김 없음.

설치 방법

30초면 됩니다. 추가 설치 없음.

  1. 위의 "Windows 다운로드" 버튼 클릭 (110MB)
  2. 원하는 폴더에 압축 해제 (예: C:\wumx\)
  3. wumx.exe 더블클릭
  4. 왼쪽 사이드바에서 워크스페이스 관리, 터미널 사용 시작
